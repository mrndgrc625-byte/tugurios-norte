# Tugurios del Norte

Narrativa digital sobre la extinción de los bares históricos de Tijuana.

Optimizado para desktop. Se recomienda visualizar en pantalla completa.

## URL
https://mrndgrc625-byte.github.io/tugurios-norte/

## Stack
- GSAP + ScrollTrigger
- JS
- HTML / CSS

## Estructura
- `index.html` — estructura principal
- `styles.css` — estilos
- `main.js` — animaciones y scroll
- `assets/` — imágenes, videos, fuentes

## Animaciones destacadas
- Mapa con hotspots
- Galeria de imágenes con pop ups

1. 4 Amigos
2. La Leyenda Negra
